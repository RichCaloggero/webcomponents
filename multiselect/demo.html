Multiselect Demo

Keyboard and Screen Reader Support

  • tab to the widget (label is announced, input element focused, screen reader enters application mode)
  • type to filter list (focus remains in input and count of items in resulting list is announced by screen reader as it changes
  • backspace to delete a character (count of number of list items announced as it changes)
  • downArrow to open and begin navigating the list
  • once in list (home, end, up, and down work as expected)
  • space to toggle a selection
  • escape to cancel current selection and return focus back to input element
  • when focused on input element, tab to the currently selected group (screen reader announces group label followed by currently selected item, or placeholder if no selection
  • change event fired on "x-multiselect" element when item's selected state changes
